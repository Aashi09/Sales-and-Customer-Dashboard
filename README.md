## üß≠ Project Objective

The goal of this project is to create a comprehensive sales dashboard that helps sales managers, executives, and analysts understand:

- Year-over-year sales performance  
- Monthly sales and profit trends  
- Weekly trends  
- Subcategory performance comparisons  
- Customer behavior and top-profit customers  

The project was built using a BI-style workflow, including requirement analysis, mockups, data preparation, chart building, and final dashboard design.

---

## üõ†Ô∏è Steps Followed (Development Process)

### **1. Analyze Requirements**
- Gather dashboard objectives  
- Choose the correct visualizations  
- Create mockups  
- Select color palette  

### **2. Build the Data Source**
- Connect to data  
- Validate field names and data types  
- Build relationships / data model  
- Understand calculations needed  

### **3. Build Charts**
- Create calculated fields  
- Build individual charts (KPIs, trends, weekly metrics, comparisons)  
- Format charts (colors, padding, grid removal, tooltips)  

### **4. Build the Dashboard**
- Create layout containers  
- Place charts in structured sections  
- Apply consistent formatting  
- Add filters and interactions  
- Add icons
